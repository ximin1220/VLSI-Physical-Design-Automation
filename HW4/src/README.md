##  How to Compile

1. Go to the directory **`HW4/src/`**.

2. Run the following command to compile the project:

   ```bash
   $ make
   ```

   After compilation, the executable file **`hw4`** will be generated in **`HW4/bin/`**.

3. To remove the executable and all intermediate files, use:

   ```bash
   $ make clean
   ```

---

##  How to Run

### Usage:

```bash
./hw4 <input.txt> <output.txt>
```

### Example:

When you are in the **`HW4/bin/`** directory, run:

```bash
$ ./hw4 ../testcase/public4.txt  ../output/output.txt
```
