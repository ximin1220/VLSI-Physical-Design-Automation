##  How to Compile

1. Go to the directory **`HW3/src/`**.

2. Run the following command to compile the project:

   ```bash
   $ make
   ```

   After compilation, the executable file **`hw3`** will be generated in **`HW3/bin/`**.

3. To remove the executable and all intermediate files, use:

   ```bash
   $ make clean
   ```

---

##  How to Run

### Usage:

```bash
./hw3 <input.lef> <input.def> <output.def>
```

### Example:

When you are in the **`HW2/bin/`** directory, run:

```bash
$ ./hw3 ../testcase/public4.lef ../testcase/public4.def ../output/output.def

```
